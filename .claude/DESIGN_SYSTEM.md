# Design System - El Norte del País

## Quick Reference

### Colors
```
Primary:   #E3120B (Economist Red)
Secondary: #002D63 (Navy Blue)
Dark:      #1a1a1a (Text)
Gray:      #6b6b6b (Secondary text)
Light:     #f5f5f5 (Backgrounds)
```

### Typography
```
Display:   font-display (Playfair Display)
Serif:     font-serif (Merriweather)
Sans:      font-sans (Inter)
```

### Usage
- Headlines → `font-display`
- Body text → `font-serif`
- Labels/UI → `font-sans`

### Key Classes
```css
text-[#E3120B]      /* Red links/CTAs */
text-[#002D63]      /* Blue accents */
text-[#1a1a1a]      /* Main text */
text-[#6b6b6b]      /* Secondary */
bg-[#f5f5f5]        /* Light bg */
```

### Components
- Header: Barra roja + logo centrado
- Navigation: Mayúsculas bold
- Cards: Bordes + serif headlines
- Footer: Dark (#1a1a1a) + red border
- Forms: Borde negro + CTA rojo

### Inspiration
The Economist editorial design
- Traditional but modern
- Clear hierarchy
- Premium feel
- Readable serif
